# Front-end na Vida Real

Repositório criado para armazenar os códigos, exercícios e projetos desenvolvidos durante o curso **Front-end na Vida Real**, ministrado por Afonso Pacifer.

## Sobre o projeto

Este repositório acompanha minha evolução nos estudos de desenvolvimento front-end.

Durante as aulas, são praticados conceitos de:

* HTML;
* CSS;
* JavaScript;
* organização de projetos;
* automação de tarefas;
* pré-processadores e ferramentas do ecossistema front-end.

## Conteúdos estudados

Entre os conteúdos presentes no repositório estão:

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/PaulinhaFigueiredodev/Front-end_na_vida_real.git
```

Entre na pasta do projeto:

```bash
cd Front-end_na_vida_real/csshortcut-app
```

Instale as dependências:

```bash
npm install
```

Instale o Gulp CLI globalmente:

```bash
npm install -g gulp-cli
```

Inicie o servidor de desenvolvimento:

```bash
gulp server
```

Para gerar os arquivos finais do projeto:

```bash
gulp build
```
